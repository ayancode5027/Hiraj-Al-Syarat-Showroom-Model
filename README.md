## 📖 Overview

**Hiraj Al Sayarat Al Jadeed** is a next-generation car showroom management platform designed to streamline vehicle inventory management, customer engagement, sales operations, and showroom administration.

Built with **Python**, **Streamlit**, and **SQLite**, the system provides an intuitive and professional interface for both showroom staff and customers.

The platform supports both **new and used vehicles**, enabling dealerships to efficiently manage stock, generate leads, schedule test drives, and improve customer satisfaction.

# ⚙️ Installation Guide

### Clone Repository

```bash
git clone https://github.com/your-username/HirajAlSayarat.git
```

### Move Into Project Directory

```bash
cd HirajAlSayarat
```

### Install Required Packages

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
